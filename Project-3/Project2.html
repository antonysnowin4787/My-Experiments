YouTube Clone

A YouTube UI clone built with pure HTML and CSS. Features a sliding sidebar, centered search bar, and the YouTube logo — all without a single line of JavaScript.

Tech Used HTML5 · CSS3
Type UI Clone

What I Learned

  • CSS-only sidebar toggle using a hidden checkbox
  • The ~ general sibling selector
  • z-index layering — overlay, sidebar, header stacking
  • CSS transitions and cubic-bezier easing
  • Centering with position: absolute + translateX(-50%)
  • Grouping flex children with wrapper divs

Biggest Problem I Faced

Moving the hamburger button to the left kept breaking the layout. The fix was wrapping the button and logo together in a header-left div — two elements behaving as one flex child.
