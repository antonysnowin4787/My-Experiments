Profile Page

A multi-page profile website featuring Elon Musk, Tony Stark, and Bruce Wayne. Each person gets their own page with quotes on discipline and consistency.

Tech Used HTML5 · CSS3
Type Multi-page Site

What I Learned

  • How to share one CSS file across multiple HTML pages
  • Flexbox layouts and wrapping cards responsively
  • Media queries from mobile to desktop
  • Semantic HTML — article, figure, figcaption, nav
  • Using min() for fluid widths

Biggest Problem I Faced

Cards were aligning to the left instead of centering on wider screens. The fix was adding justify-content: center to the flex container — a small line that took me a while to find!
